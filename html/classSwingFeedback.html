void SwingFeedback::onLevel ( const char *  level)

Receives a feedback level, stores the highest pending level, and starts the worker if needed.

Level dropped to zero — swing has ended. Fire the worker now so it plays the actual peak that was accumulated, not just the first edge.

Accumulate the highest level reached during the swing. Do NOT spawn the worker yet — wait for the swing to end (rank == 0 above).

Internal worker for applying queued feedback updates.

Runs the feedback output, updates the busy state, and calls the reset callback when finished.

Clear any level that accumulated during playback so it doesn't contaminate the next swing.
