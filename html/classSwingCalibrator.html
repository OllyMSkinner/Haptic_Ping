Detailed Description

SOLID Principles: S - This file has exactly one function - to remove the gravitational bias from the acceleration value of each axis. I - The accumulate() is kept private as it is only used for moving average calibration, other caller doesn't need it. D - The std::function, which is an abstraction, allows the SwingCalibrator to do its job without needing to know the caller or the results. Manages gravity-bias calibration for the swing detector.

Usage:

  1. Construct and set a callback.
  2. Call feed() on every IMU sample.
    • While calibrating, feed() returns true (caller should skip measurement).
    • Once done, feed() returns false and the bias is ready via biasX/Y/Z().
  3. Call triggerRecal() whenever the position detector confirms the start position. The next RECAL_SAMPLES samples will be used to refresh the bias.
