\doxysubsection{Detailed Description}
SOLID Principles\+: S -\/ This file has exactly one function -\/ to remove the gravitational bias from the acceleration value of each axis. I -\/ The \doxylink{classSwingCalibrator_a7965dffc00c681bf30ec0ac24a652f2c}{accumulate()} is kept private as it is only used for moving average calibration, other caller doesn\textquotesingle{}t need it. D -\/ The std\+::function, which is an abstraction, allows the \doxylink{classSwingCalibrator}{Swing\+Calibrator} to do its job without needing to know the caller or the results. Manages gravity-\/bias calibration for the swing detector.

Usage\+:
\begin{DoxyEnumerate}
\item Construct and set a callback.
\item Call \doxylink{classSwingCalibrator_a763b2ac04244cc6246a8b011e6727754}{feed()} on every IMU sample.
\begin{DoxyItemize}
\item While calibrating, \doxylink{classSwingCalibrator_a763b2ac04244cc6246a8b011e6727754}{feed()} returns true (caller should skip measurement).
\item Once done, \doxylink{classSwingCalibrator_a763b2ac04244cc6246a8b011e6727754}{feed()} returns false and the bias is ready via bias\+X/\+Y/Z().
\end{DoxyItemize}
\item Call \doxylink{classSwingCalibrator_af3698c7f929fb86fe97e7c8337b6213d}{trigger\+Recal()} whenever the position detector confirms the start position. The next RECAL\+\_\+\+SAMPLES samples will be used to refresh the bias. 
\end{DoxyEnumerate}
