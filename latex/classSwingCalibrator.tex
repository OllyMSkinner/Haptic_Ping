\doxysubsection{Detailed Description}
Gravity-\/bias calibrator for the swing detector. Feed IMU samples continuously; while calibrating, \doxylink{classSwingCalibrator_a763b2ac04244cc6246a8b011e6727754}{feed()} returns true. Once ready, bias\+X/\+Y/Z() hold the current bias estimate. \doxylink{classSwingCalibrator_af3698c7f929fb86fe97e7c8337b6213d}{trigger\+Recal()} starts a short recalibration window.
