\doxysubsection{Detailed Description}
SOLID principles\+: S -\/ \doxylink{classSwingDetector}{Swing\+Detector} categorised the swing speed intro 3 levels, and creates a callback when the level changes. This actually does 2 things but it is very small so we decided to wrap them up with each other. I -\/ The calculation of the level classification in made private as other caller doesn\textquotesingle{}t need to know. D -\/ The std\+::function is an abstraction which allows for any inputs to the function. Classifies linear acceleration magnitude into duty cycle levels. Fires the registered callback only when the level changes.
