\doxysubsection{Detailed Description}
raised to reject motor EMI noise floor 

Gate logic\+:
\begin{DoxyEnumerate}
\item When the correct start position is held, is\+Armed\+\_\+ becomes true and the bias is recalibrated. The system must still be in this valid position when the piezo is triggered.
\item When the piezo is pressed while armed, is\+Active\+\_\+ becomes true and the motor feedback starts. Once active, any further piezo events are ignored to avoid false triggers caused by motor vibration.
\item When the \doxylink{classSwingFeedback}{Swing\+Feedback} timer ends, \doxylink{classSwingProcessor_aea9cf6f18b80fb71de6eb85e570ccf3e}{reset()} is called to clear both state flags. Declares the swing processor class, including its callback types and main functions for handling force, position, reset, and sensor sample processing. 
\end{DoxyEnumerate}
